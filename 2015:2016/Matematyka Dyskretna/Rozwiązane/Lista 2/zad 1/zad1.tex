\begin{thm}
    Niech $a$ będzie liczbą niewymierną, a $n$ liczbą całkowitą dodatnią. Pokaż, że 
    $$\lfloor an \rfloor + \lfloor (1-a)n \rfloor = n-1$$
\end{thm}

\begin{proof}
Wykonajmy następujące przekształcenia arytmetyczne:

$$\lfloor an \rfloor + \lfloor (1-a)n \rfloor = \lfloor an \rfloor + \lfloor n - an \rfloor = 
\left( \lfloor an \rfloor + \lfloor - an \rfloor \right) + n =  \left( \lfloor an \rfloor - \lceil  an \rceil \right) + n = n-1$$

\end{proof}

\begin{remark}
Skorzystałem tutaj z własności $\lfloor - an \rfloor = - \lceil  an \rceil$, która jest oczywista, bo $a$ jest niewymierne. Jednak warto wspomnieć, że założenie o niewymierności jest w którymś miejscu dowodu potrzebne.
\end{remark}

\begin{remark}
Jak wygląda odpowiednik powyższej równości dla powały?
$$\lceil an \rceil + \lceil (1-a)n \rceil = n+1$$

\begin{proof}
$$
\lceil an \rceil + \lceil (1-a)n \rceil = \lceil an \rceil + \lceil n - an \rceil = \lceil an \rceil + n + \lceil - an \rceil  = n + \lceil an \rceil - \lfloor an \rfloor = n+1
$$
\end{proof}

\end{remark}
